### 🧐 关于我

“比起堆技术名词，我更习惯把真正用过、踩过坑、也值得复用的东西做成开源仓库，持续迭代下去。”

- 🔧 **核心技术栈**：熟练掌握 **STM32** (H7/F1)、**MSPM0** 及 **沁恒 CH系列** 单片机软硬件开发。
- 🧮 **算法与信号处理**：热衷于电赛信号题分析。在编写 FFT/IFFT、窗函数等 DSP 算法时，坚持代码规范化与高效复用（如统一将点数指定为 `FFT_N` 宏定义）。
- ⚙️ **底层开发习惯**：比起浮于表面的性能调优，我更倾向于从总线矩阵与存储器拓扑切入。
- 💻 **工具与自动化**：C / Python | 熟练配置 Keil MDK, CLion, VS Code | 习惯编写自动化脚本（如基于 OpenCV 批量处理特定文件夹下的 JPEG 图片）提升开发流效率。
- 💬 **常聊的话题**：嵌入式底层架构、单片机踩坑记录、电赛信号题方案、DSP 算法实现。
